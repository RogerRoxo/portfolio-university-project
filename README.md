# Personal Portfolio Website

A simple personal portfolio website built with HTML, CSS, and JavaScript as a college project.

## Pages

- **About Me** — A brief introduction and personal overview  
- **Education** — Academic background, skills, and learning journey  
- **Portfolio** — Projects, work, and achievements  
- **Contact** — Ways to get in touch  

## Technologies Used

- HTML5  
- CSS3  
- JavaScript  

## Features

- Multi-page layout  
- Clean and responsive design  
- Easy navigation  
- Simple and organized structure  

## Purpose

This project was developed as a college assignment to create a professional personal portfolio website, showcasing personal information, education, projects, and contact details while practicing front-end web development skills.
